If you haven’t already, make an account! This will be a tutorial to set up a Trello Board to be able to track your different tasks.

Click on “Create”

Now take the time to create your own test plan for Employee manager, moving everything into Trello, including the issues you ran into while doing some testing.

Each issue you list should be it’s own card. And after you’ve created each card, your next task is to add traceability between cards.

To do this, you can add an attachment to a card in Trello:

You can then search for another card on your board, and link the two cards together.

It’s good to have clean traceability so you can always find what you’re looking for!

Save a link to your trello board in a text file and push that file to your GitHub account so that your education team can review your board.
